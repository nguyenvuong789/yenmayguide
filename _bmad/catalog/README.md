# Yen May Vintage BMad Catalog

This project-specific BMad catalog is for launching and operating Yen May Vintage across 5 channels: Facebook, Instagram, TikTok, Threads, and Google Maps.

Source priority:

1. `Yên Mây Store.pdf` - brand source of truth.
2. `YenMay_Marketing_Plan_2026_Vin.md` - execution plan and channel strategy.
3. `Đánh Giá Plan Marketing — Yên Mây Vintage 2026.md` - critique, gaps, and launch risk controls.

The active BMad Help catalog lives at:

`_bmad/_config/bmad-help.csv`

Recommended workflow:

1. `brand` - create `brand-foundation.md`.
2. `content-strategy` - create `content-strategy.md`.
3. `social` - create `agency-production-brief.md`.
4. Review `agency-production-brief.md` before sending it to the agency:
   - `social` - create `agency-production-brief.review-social.md`.
   - `marketing-psychology` - create `agency-production-brief.review-psychology.md`.
   - `copywriting` - create `agency-production-brief.review-copy.md`.
   - Optional: `bmad-review-adversarial-general` - create `agency-production-brief.review-adversarial.md`.
5. `social` - create `social-launch-plan.md` for Facebook, Instagram, TikTok, Threads, and Google Maps.
6. `image` - create `visual-asset-plan.md`.
7. `copywriting` - create `page-copy.md`.
8. `analytics` - create `measurement-plan.md`.
9. `ads` and `ad-creative` - use only after organic posts show traction.

Operational rule:

Use the PDF for identity, tone, color, concept, audience, and spirit. Use the marketing plan for channel tactics, keyword phases, content pillars, cadence, and KPIs. Use the evaluation document to tighten weak points before launch: agency briefing, Facebook launch, urgency, retention, manual tracking, and CTA split by audience.

## Agency Brief Review Workflow

Use this workflow whenever `_bmad/outputs/agency-production-brief.md` changes materially.

1. Social execution review:
   - Skill: `social`
   - Output: `_bmad/outputs/agency-production-brief.review-social.md`
   - Check: platform fit, shot list clarity, cross-posting, agency handoff, production feasibility, and whether it prevents generic fashion-shop content.

2. Buyer psychology review:
   - Skill: `marketing-psychology`
   - Output: `_bmad/outputs/agency-production-brief.review-psychology.md`
   - Check: Activation Energy, proof, trust, urgency, ethical scarcity, CTA clarity, Vietnamese DM path, foreign visitor path.

3. Copy review:
   - Skill: `copywriting`
   - Output: `_bmad/outputs/agency-production-brief.review-copy.md`
   - Check: hooks, captions, overlays, CTAs, DM prompts, clarity, specificity, and whether the brief gives agency usable words.

4. Optional adversarial review:
   - Skill: `bmad-review-adversarial-general`
   - Output: `_bmad/outputs/agency-production-brief.review-adversarial.md`
   - Check: failure modes, ambiguity, missing owners, unrealistic assumptions, and ways the agency could misread instructions.

Review outputs should produce specific patch recommendations, not just opinions. After the brief is patched, each review should also keep a resolution summary so the audit trail stays readable.
